Hardware and Software Architecture

Project ENIGMA's hardware setup centers on simplicity and responsiveness. A rotary encoder allows precise rotor control, while tactile push buttons manage selection and configuration. A 16x2 LCD simulates the Enigma lampboard, giving real-time feedback on user inputs.

  • Rotary Encoder: Rotor stepping and character selection
  • Push Buttons: For rotor control, reset, and config
  • LCD Display: Simulated lampboard output
  • Arduino Nano: Core microcontroller
  • Serial Interface: For GUI integration

üß† Software Design

The Enigma logic is modeled using a modular state machine architecture. We implemented historical rotor behavior, plugboard swaps, and reflector logic. Input threads manage encoder and button polling, while output threads handle display and communication.

  • Rotor Stepping & Double-Stepping: Fully simulated
  • Plugboard Logic: Real-time letter swapping
  • Reflector Circuit: Historical symmetry emulation
  • Threading: Protothreads used for concurrency
  • Python GUI: Serial monitoring and control
